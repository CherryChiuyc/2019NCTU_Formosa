Education Kit

Let’s race!!!!!!!!!!!

   The goal of our education kit is to get most people be aware of the gene mutation, and show how different mutagenicity could be visualized.

   “” is the product we developed. A car racing IoT game combined with mutagenicity predicting system. All users have to do is connect the IoT model cars to internet and get on to our web page with a searching system, type in the name of the chemical compound.

Steps for Playing the Racing Game:

1. Connect the model cars to the internet.

2. Go the following link.

3. Type in the name of the chemical compound (type to if you want to race).

4. Press “submit.”

5. Watch the racing game!

   The model cars are carrying the Arduino Yun mini, an Arduino broad with wifi connection. With the burned in code, it receives the signal from the IoTTalk(see more info on “IoTTalk”).

Website

   On the other hand, there is a mutagenicity predicting system on the website. As our dry lab team developed the data set, a python-based machine learning was built in an elegant way. This is how the magic happened:

   1. Users type in the name of the compound.

   2. The name of the compound will be transformed into SMILE form.

   3. The SMILE file is sent in to the data set with the predicting formula.

   4. The mutagenicity output will go through a linear transformation then be sent to
    IoTTalk.

   5. The cars will receive the output and Runnnnnnn!
